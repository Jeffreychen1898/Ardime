# Ardime

A simple rendering library

# Navigating The Project

**Run The Code**

* Install the npm modules with the following commands: (If not already done so)

  * ```
    cd test
    npm install
    ```

* Run the project

  * ```
    cd test
    npx webpack build
    node index
    ```

**Compress The Library**

```
cd test
npx webpack build
```
